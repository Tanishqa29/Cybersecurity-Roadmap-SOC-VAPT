## 🔹 Network Devices

### ▪️ Hub
A hub is a basic networking device that connects multiple devices in a network. It broadcasts incoming data to all connected devices without checking the destination. It works on the Physical Layer (Layer 1).

---

### ▪️ Switch
A switch connects devices in a LAN and sends data only to the intended device using MAC addresses. It improves network efficiency and reduces collisions. It works on the Data Link Layer (Layer 2).

---

### ▪️ Router
A router connects multiple networks and routes data between them using IP addresses. It decides the best path for data to travel. It is commonly used to connect a home network to the Internet.

---

### ▪️ Modem
A modem converts digital signals to analog and vice versa to enable internet communication. It connects your network to the ISP.

---

### ▪️ Access Point (AP)
An access point provides wireless connectivity (WiFi) to devices. It acts as a bridge between wired and wireless networks.

---

### ▪️ Repeater
A repeater boosts or regenerates weak signals in a network. It helps extend the distance of the network.

---

### ▪️ Bridge
A bridge connects two LAN segments and reduces traffic by filtering data based on MAC addresses. It works on the Data Link Layer.

---

### ▪️ Gateway
A gateway connects different types of networks and acts as a translator between them. It can operate on multiple OSI layers.

---

### ▪️ Firewall 🔐
A firewall is a security device that monitors and controls incoming and outgoing traffic. It blocks unauthorized access and allows legitimate communication.

---

### ▪️ NIC (Network Interface Card)
A NIC is a hardware component that allows a computer to connect to a network. It provides physical access to the network and uses a MAC address for identification. It works on Layer 1 and Layer 2 of the OSI model.

---

## 🔥 Hub vs Switch (Important Difference)

| Feature       | Hub                           | Switch                          |
|--------------|--------------------------------|----------------------------------|
| Working Layer | Physical Layer (Layer 1)       | Data Link Layer (Layer 2)        |
| Data Transfer | Sends data to all devices      | Sends data to specific device    |
| Intelligence  | Dumb device                    | Smart device                     |
| Speed         | Slow                           | Fast                             |
| Collision     | More collisions                | Very less collisions             |
| MAC Address   | Not used                       | Uses MAC address table           |
