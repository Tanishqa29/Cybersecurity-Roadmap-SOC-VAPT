# Episode 06 - SIEM and Security Operations Center (SOC)

## Overview

A Security Operations Center (SOC) monitors, detects, investigates, and responds to cyber threats.

SIEM collects and analyzes logs from multiple sources.

---

## Log Sources

- Firewalls
- Servers
- Endpoints
- Active Directory
- Cloud Services
- Applications

---

## SIEM Workflow

Collect Logs

↓

Normalize Data

↓

Correlation Rules

↓

Alert Generation

↓

Investigation

↓

Incident Response

---

## Common SIEM Platforms

- Splunk
- Microsoft Sentinel
- IBM QRadar
- Wazuh

---

## SOC Responsibilities

- Threat Monitoring
- Alert Investigation
- Incident Response
- Threat Hunting
- Reporting

---

## Key Takeaways

A SOC provides continuous visibility into enterprise security and enables rapid threat detection.
