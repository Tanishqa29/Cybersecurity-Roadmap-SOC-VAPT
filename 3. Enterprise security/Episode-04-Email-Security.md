# Episode 04 - Enterprise Email Security

## Overview

Email remains the most common attack vector used by cybercriminals.

---

## Common Attacks

- Phishing
- Spear Phishing
- Business Email Compromise (BEC)
- Malware Attachments
- Credential Theft

---

## Security Technologies

- SPF
- DKIM
- DMARC
- Secure Email Gateway
- Anti-Spam Filtering
- Sandboxing

---

## Enterprise Scenario

An employee receives an invoice attachment.

The secure email gateway scans the file and detects ransomware before it reaches the user's inbox.

---

## Best Practices

- Conduct phishing awareness training
- Enable MFA
- Verify suspicious requests
- Block executable attachments
- Monitor email logs

---

## Key Takeaways

Email security combines technology and user awareness to reduce phishing-related incidents.
