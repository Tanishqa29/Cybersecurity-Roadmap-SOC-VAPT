# Episode 03 - Endpoint Security

## Overview

Endpoints include laptops, desktops, mobile devices, and servers connected to the enterprise network.

Every endpoint can become an entry point for attackers.

---

## Common Threats

- Malware
- Ransomware
- Phishing
- USB Attacks
- Insider Threats

---

## Protection Methods

- Antivirus
- Endpoint Detection and Response (EDR)
- Disk Encryption
- Patch Management
- Device Control

---

## Enterprise Example

A phishing email infects one employee's laptop.

An EDR solution detects unusual activity and isolates the infected device before malware spreads.

---

## Best Practices

- Keep systems updated
- Enable endpoint monitoring
- Restrict USB devices
- Use application whitelisting
- Encrypt sensitive devices

---

## Key Takeaways

Strong endpoint security reduces the likelihood of attackers gaining access to enterprise systems.
